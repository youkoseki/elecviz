# elecviz

衆議院選挙（小選挙区）の結果を可視化するページです。

**[https://youkoseki.github.io/elecviz/](https://youkoseki.github.io/elecviz/)**

## 概要

2021年（第49回）、2024年（第50回）、2026年（第51回）の衆議院議員総選挙について、小選挙区ごとの得票数を積み上げ棒グラフで表示します。

## 見方

- 都道府県をドロップダウンか地図から選択
- 選挙区ごとのパーマリンク対応
- 棒グラフは得票数の多い候補者が下から順に積み上げ
- 各候補者のラベルは「名前 / 政党 得票率% / 当選🎉 or 復活🔄」
- **前職**（前回当選した現職候補）は**太字**で表示
- 縦軸（得票数）のスケールは全選挙区で統一されているため、選挙区間の得票数の大小をそのまま比較可能

## データ

[矢内勇生氏が公開している選挙データ](https://yukiyanai.github.io/jp/resources/)を利用しています。
